# Projeto de E-Commerce
Interface de E-Commerce implementada para uma Atividade da disciplina de Programação Orientada a Objetos. Este repositório consiste da unificação colaborativa das partes de todas as equipes da turma.

## Como contribuir?
1. Clone o repositório na sua workspace do eclipse com `git clone https://github.com/kprincipe/ecommerce-poo`
2. Crie uma nova branch para sua implementação com `git checkout -b metodo` onde `metodo` é o nome do método da sua equipe.
3. Faça as modificações e verifique se tudo funciona corretamente!
4. Adicione os arquivos modificados com `git add .`
5. Comite as mudanças com uma mensagem descritiva sobre seu método com `git commit -m "mensagem"`.
6. Faça push das mudanças para o repositório com `git push -u origin main`.
